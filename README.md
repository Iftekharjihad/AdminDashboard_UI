# Simple Inventory Management System

A web-based application built with **C#** and **ASP.NET Core MVC** that allows users to manage products through a secure dashboard. This project demonstrates the implementation of the MVC architectural pattern, Entity Framework Core, and User Authentication.

## 🚀 Features

* **User Authentication:** Secure Login and Registration system to protect the dashboard.
* **Product Dashboard:** A centralized view of all inventory items.
* **Full CRUD Operations:**
    * **Add:** Create new products with details.
    * **Edit:** Update existing product information.
    * **Delete:** Remove products from the database.
    * **View:** Detailed list view of all available products.
* **Responsive UI:** Built with Bootstrap for a clean look on all devices.

## 🛠️ Tech Stack

* **Framework:** ASP.NET Core MVC
* **Language:** C#
* **Database:** SQL Server (via Entity Framework Core)
* **Frontend:** Razor Pages, HTML5, CSS3, Bootstrap
* **Tools:** Visual Studio, NuGet Package Manager
